# 🌱 Agro Forte - Futuro Sustentável

> **Projeto Educacional desenvolvido para o Programa Agrinho** > Uma plataforma interativa para conscientizar e ensinar sobre a importância do agronegócio moderno, sua cadeia produtiva e as práticas sustentáveis que movem o Brasil.

## 🎯 Objetivo (Tema Agrinho)
O principal objetivo deste projeto é levar informação clara, visual e interativa sobre o **Agronegócio Sustentável**. Alinhado com as diretrizes do Programa Agrinho, o site busca quebrar mitos sobre o campo, mostrando como a tecnologia, a inovação e a responsabilidade ambiental caminham juntas para garantir a segurança alimentar do planeta e a preservação dos nossos recursos naturais.

## 🚀 Funcionalidades
* Navegação Fluida (Smooth Scroll): Menu responsivo que leva o usuário diretamente para as seções da página sem recarregá-la.
* Modais Interativos (Pop-ups): Cartões clicáveis nas seções "Sobre" e "Cadeia Produtiva" que exibem explicações detalhadas de forma dinâmica.
* Gráficos Dinâmicos: Renderização de dados reais sobre a produção e exportação do agronegócio brasileiro usando gráficos de barras e rosca (*doughnut*).
* Design Responsivo e Temático: Identidade visual baseada em tons de verde, totalmente adaptável para dispositivos móveis, tablets e computadores.

## 🛠 Tecnologias Utilizadas
O projeto foi construído utilizando tecnologias web fundamentais (HTML, CSS e JavaScript), sem a necessidade de frameworks complexos, garantindo leveza e rapidez no carregamento:

* HTML: Estruturação semântica de todo o conteúdo e seções.
* CSS: Estilização personalizada, uso de variáveis, efeitos de *hover*, transições suaves e layout responsivo.
* JavaScript (Vanilla JS):** Lógica para abertura/fechamento dos modais interativos.
  
## 📂 Estrutura do Projeto
Para que o site funcione corretamente, os arquivos devem estar organizados na mesma pasta (diretório raiz) da seguinte forma:
├── index.html       # Estrutura principal da página (HTML)
├── style.css        # Estilização visual e responsividade (CSS)
└── script.js       # Comportamento dos modais e configuração dos gráficos (JS)
